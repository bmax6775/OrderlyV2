OrderDesk Installation

Quick & Easy Setup for NameCheap Shared Hosting

1

Download OrderDesk Files

Download all project files and create a ZIP archive for upload.

Ready: You should have all OrderDesk files in a ZIP file ready for upload.
2

Access Your NameCheap cPanel

  1. Login to your NameCheap account
  2. Go to Hosting List → Manage
  3. Click cPanel button
  4. Find and click File Manager
3

Upload OrderDesk Files

  1. Navigate to public_html folder
  2. Click Upload button
  3. Select your OrderDesk ZIP file
  4. After upload, right-click the ZIP → Extract
  5. Move all files to public_html root
  6. Delete the ZIP file
Important: All files must be in public_html root, not in a subfolder.
4

Create MySQL Database

In cPanel, click MySQL Databases and follow these steps:

Create Database:
Database Name: yourusername_orderdesk
Click "Create Database"
Create Database User:
Username: yourusername_admin
Password: [Create a strong password]
Click "Create User"
Add User to Database:
Select your database and user
Grant: ALL PRIVILEGES
Click "Add"
5

Import Database Schema

  1. In cPanel, click phpMyAdmin
  2. Select your OrderDesk database from left sidebar
  3. Click Import tab
  4. Choose file: database_mysql.sql
  5. Click Go
  6. Wait for success message
6

Configure Database Connection

Edit config.php file and update these settings:

$db_host = 'localhost';
$db_name = 'yourusername_orderdesk'; // Your database name
$db_user = 'yourusername_admin'; // Your database user
$db_pass = 'your_password_here'; // Your database password
7

Set File Permissions

Set proper permissions for uploads folder:

  1. Right-click uploads folder
  2. Select Permissions
  3. Set to 755
  4. Check Recurse into subdirectories
  5. Click Change Permissions
8

Test Your Installation

Visit your domain in a web browser. You should see the OrderDesk login page.

Test with Demo Accounts:
Role Username Password
Super Admin superadmin password
Admin demoadmin password
Agent demoagent password
9

Secure Your Installation

Immediate Actions:
  • Change Super Admin password
  • Create your own admin account
  • Delete demo accounts
  • Enable SSL certificate
Optional Enhancements:
  • Update branding & colors
  • Add your team members
  • Import your stores
  • Set up backups

Common Issues & Solutions

Database Connection Error:
  • Check database name in config.php
  • Verify username and password
  • Ensure user has database privileges
File Upload Issues:
  • Check uploads folder permissions (755)
  • Verify PHP upload limits
  • Ensure sufficient disk space

Installation Complete!

Your OrderDesk system is now ready for production use. Start by creating your admin account and customizing the branding.
